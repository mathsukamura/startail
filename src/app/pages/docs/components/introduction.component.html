Bem-vindo a documentacao do StarTail Admin.

O que e o StarTail Admin?

StarTail Admin e um template administrativo open source moderno para Angular 19, inspirado nos principios de design do shadcn/ui. Ele oferece uma colecao completa de componentes reutilizaveis, uma arquitetura limpa e uma otima experiencia para desenvolvedores.

Principais Recursos

  • 50+ componentes prontos para uso
  • Tema claro e escuro com transicao suave
  • Tailwind CSS v4 com variaveis semanticas
  • Autenticacao integrada
  • Tabelas com ordenacao, filtros e paginacao
  • Graficos interativos com ApexCharts

Primeiros Passos

Para comecar a usar o StarTail Admin, siga o guia de instalacao. Depois, explore os componentes disponiveis na barra lateral para ver exemplos e codigos de uso.
